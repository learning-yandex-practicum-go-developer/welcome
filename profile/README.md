### 🛡️ Инструкция для ревьюеров / Reviewer Guide

Для получения доступа к проверке приватного репозитория, пожалуйста, следуйте алгоритму:

1. **Предоставьте свой GitHub-ник** (username).
2. Я добавлю вас в список **Collaborators**.
3. После подтверждения ссылка на Pull Request станет доступна для ревью.
*Куратор уведомлен о данном порядке.*

> **For Reviewers:** To access a private repository, please provide your GitHub username. Once added as a collaborator, the PR link will become accessible.
